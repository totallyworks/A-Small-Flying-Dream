S - positive/negative

XXX - value of the command

Last character of the command - #

B - boolean value of the command


| Command  | Description                                                          |
| -------- | -------------------------------------------------------------------- |
| ltSXX#   | Set left aileron's trim                                              |
| rtSXX#   | Set right aileron's trim                                             |
| etSXX#   | Set elevator's trim                                                  |
| dtSXX#   | Set rudder's trim                                                    |
| lsSXXXX# | Set left aileron's scale value                                       |
| rsSXXXX# | Set right aileron's scale value                                      |
| esSXXXX# | Set elevator's scale value                                           |
| dsSXXXX# | Set rudder's right-side scale value                                  |
| fsSXXXX# | Set rudder's left-side scale value                                   |
| lmXXX#   | Move left aileron                                                    |
| rmXXX#   | Move right aileron                                                   |
| emXXX#   | Move elevator                                                        |
| dmXXX#   | Move rudder                                                          |
| mmB#     | Toggle throttle maximization                                         |
| smXXX#   | Set throttle scale value                                             |
| yoB#     | Turn IMU on/off                                                      |
| ysB#     | Toggle Roll Stabilization mode                                       |
| ypB#     | Toggle Pitch Stabilization mode                                      |
| ym#      | Write time since power-on to SD card                                 |
| yaB#     | Toggle accelerometer calibration                                     |
| ygB#     | Toggle magnetometer calibration                                      |
| yz#      | Reset the I-term of the roll and pitch stabilization PID controllers |
| pc#      | Calibrate airspeed sensor                                            |
| bp#      | Calibrate pressure sensor                                            |
| bt#      | Calibratetemperature sensor                                          |
| baB#     | Toggle Altitude Hold mode                                            |
| bsXXXXX# | Set Altitude Hold mode's altitude setpoint                           |
| piSXXXX# | Set Roll Stabilization mode's P coefficient                          |
| iiSXXXX# | Set Roll Stabilization mode's I coefficient                          |
| diSXXXX# | Set Roll Stabilization mode's D coefficient                          |
| oiSXXXX# | Set Pitch Stabilization mode's P coefficient                         |
| kiSXXXX# | Set Pitch Stabilization mode's P coefficient                         |
| ciSXXXX# | Set Pitch Stabilization mode's P coefficient                         |
| ri#      | Request various parameters for display in Ground Station app         |
